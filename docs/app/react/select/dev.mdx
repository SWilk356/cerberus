## Customizing

To customize the Select family we recommend extending the slot recipe provided by the `@cerberus/panda-preset` package.

## API

<NoteAdmonition description="The Select component must be used within a Field provider." />

```ts showLineNumbers=false
export interface FieldProps {
  disabled?: boolean
  invalid?: boolean
  required?: boolean
  readOnly?: boolean
}

define function Field(props: PropsWithChildren<FieldProps>): ReactNode
```

```ts showLineNumbers=false
export interface SelectProps extends SelectHTMLAttributes<HTMLSelectElement> {
  describedBy?: string
  id: string
  size?: 'sm' | 'md' | 'lg'
}

define function Select(props: SelectProps): ReactNode
```

### Props

The `SelectProps` component accepts the following props:

| Name     | Default | Description                                                   |
| -------- | ------- | ------------------------------------------------------------- |
| describedBy     |    | The `id` attribute of the FieldMessage the select is associated with. |
| id   |    |  An identifier that is shared with the Label `htmlFor` attribute.        |
| size   | `lg`   |  The size of the select field.        |
