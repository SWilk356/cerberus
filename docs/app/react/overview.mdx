# Overview

The Cerberus React library is a high performant and flexible collection of React components that are used to build the Cerberus UI fully compatible with React 19 Server Components.

<Admonition
  heading="Note"
  description="The Cerberus React library requires the use of Panda-CSS and the Cerberus Preset.
  Please be sure you have setup your project to use these libraries before using
  Cerberus React."
/>

## Installation

Install the Cerberus React library, with the package manager of your choice:

```bash
pnpm install '@cerberus-design/react'
```
