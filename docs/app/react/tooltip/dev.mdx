To use a Tooltip, simply add the `data-tooltip`, `aria-label` attributes to the element you want to add a tooltip to. Optionally, you can include `data-position` to specify the position of the tooltip.

```tsx
<button ariaLabel="Like something" data-tooltip data-position="top">
```

## Usage

<CodePreview preview={<BasicTooltipPreview />}>
```tsx title="icon-button.tsx" {6}
import { IconButton } from '@cerberus-design/react'
import { ThumbsUp } from '@cerberus/icons'

function BasicTooltipPreview() {
  return (
    <IconButton ariaLabel="Get information">
      <Information />
    </IconButton>
  )
}
```

## Customization

To customize the tooltip, you need to update the globalCss options within your Panda theme.

```ts title="globalCss.ts"
':is(div,button,span,svg)': {
  _tooltip: {
    _hover: {
      _before: {}, // bubble
      _after: {}, // arrow
    },
    _focus: {
      _before: {},
      _after: {},
    },
  },
},
```

## API

All of the following props are required to display a tooltip.

| Name     | Default | Description                                                   |
| -------- | ------- | ------------------------------------------------------------- |
| aria-label |    `undefined`     | The content of the tooltip and accessible label. |
| data-tooltip | `undefined`   | The attribute that triggers a tooltip. |
| data-position | `undefined`    | The position of the tooltip.    |
