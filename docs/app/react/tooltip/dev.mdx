To use a Tooltip, simply add the `data-tooltip` and `aria-label` attributes to the element you want to add a tooltip to.

```tsx
<button data-tooltip ariaLabel="Like something">
```

## Usage

<CodePreview preview={<BasicTooltipPreview />}>
```tsx title="icon-button.tsx" {6}
import { IconButton } from '@cerberus-design/react'
import { ThumbsUp } from '@cerberus/icons'

function BasicTooltipPreview() {
  return (
    <IconButton ariaLabel="Like something">
      <ThumbsUp />
    </IconButton>
  )
}
```
</CodePreview>

<NoteAdmonition description="IconButtons provide the data-tooltip attribute for you by default." />

## Positions

<CodePreview preview={<PositioningTooltipPreview />}>
```tsx title="icon-button.tsx" {10,13,19,23}
import { Information } from '@cerberus-design/icons'
import { IconButton } from '@cerberus-design/react'
import { hstack } from '@cerberus/styled-system/patterns'

function PositioningTooltipPreview() {
  return (
    <div className={hstack({
      gap: '10'
    })}>
      <IconButton ariaLabel="Top position" data-tooltip data-position="top">
        <Information />
      </IconButton>
      <IconButton ariaLabel="Right position" data-tooltip data-position="right">
        <Information />
      </IconButton>
      <IconButton
        ariaLabel="Bottom position"
        data-tooltip
        data-position="bottom"
      >
        <Information />
      </IconButton>
      <IconButton ariaLabel="Left position" data-tooltip data-position="left">
        <Information />
      </IconButton>
    </div>
  )
}
```
</CodePreview>

## Customization

To customize the tooltip, you need to update the globalCss options within your Panda theme.

```ts title="globalCss.ts"
':is(div,button,span,svg)': {
  _tooltip: {
    _hover: {
      _before: {}, // bubble
      _after: {}, // arrow
    },
    _focus: {
      _before: {},
      _after: {},
    },
  },
},
```

## API

There is no API for the Tooltip. The Tooltip is a simple pseudo element that uses the `data-tooltip` and `aria-label` attributes to display a tooltip.`
