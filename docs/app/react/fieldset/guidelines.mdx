## Usage

<WhenNotToUseAdmonition description="Labels should always accompany any type of input." />


## Standard Labels

Standard labels are used for most input use cases.

<CodePreview preview={<LabelOptionalPreview />} />

<WhenToUseAdmonition description="When you use any type of standard inputs within a form." />

## Hidden Labels

Hidden labels are used when the label is not visible to the user, but still needs to be present for screen readers.

<CodePreview preview={<LabelHiddenPreview />} />

<WhenToUseAdmonition description="When you need an input that doesn't use a visual label (i.e., global search input)" />

## Required Labels

Required labels are used when the input is required.

<CodePreview preview={<LabelBasicPreview />} />

<WhenToUseAdmonition description="When the input is required." />

## Placement

Labels should be placed above the input field. For checkboxes and radio buttons, the label should be placed to the right of the input.
