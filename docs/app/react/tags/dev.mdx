## Usage

You can mix and match all of the examples to create the button style of your choosing.

### Default

<CodePreview preview={<BasicTagPreview />}>

## Customization

You can customize the tag by utilizing the `cx` function.

<CodePreview preview={<CustomTagPreview />}>
```tsx title="button.tsx"
import { Button } from '@cerberus-design/react'
import { css } from '@cerberus/styled-system/css'

function CustomButton() {
  return (
    <Button
      className={css({
        bgColor: 'yellow.800',
        color: 'black',
        _hover: {
          bgColor: 'yellow.500',
        },
      })}
    >
      Wu-Tang Forever
    </Button>
  )
}
```
</CodePreview>

## API

```ts showLineNumbers=false
export interface TagProps extends HTMLAttributes<HTMLSpanElement> {
  palette?: Sentiment
  shape?: 'rounded' | 'pill'
  usage?: 'filled' | 'outline'
}
export interface ClickableTagProps extends HTMLAttributes<HTMLSpanElement> {
  shape?: 'pill'
  onClick: () => void
  usage?: 'filled'
}

define function Tag(props: TagProps | ClickableTagPRops): ReactNode
````

### Props

The `Tag` component accepts the following props:

| Name     | Default | Description                            |
| -------- | ------- | -------------------------------------- |
| palette  | `action`   | The color palette of the tag. |
| usage    | `filled`    | The style treatment of the tag.    |
| shape    | `rounded`    | The shape of the tag.              |
| onClick  | `undefined` | How to create a Closable tag.         |
