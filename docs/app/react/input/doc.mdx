# Input

An accessible input field to allow the user to enter text into the UI.

```ts
import { Field, Input } from '@cerberus-design/react'
```

<WhenToUseAdmonition description="When you need a way for the user to enter text into the UI." />

## Usage

<CodePreview preview={<InputBasicPreview />}>
```tsx title="first-name-input.tsx" {7}
import { Field, Label } from '@cerberus-design/react'

function InputBasicPreview() {
  return (
    <Field required>
      <Label htmlFor="first_name">First Name</Label>
      <input id="first_name" type="text" />
    </Field>
  )
}
```
</CodePreview>

<WhenNotToUseAdmonition description="In order to ensure a form element is accessible, the Label should always be used." />

## API

```ts showLineNumbers=false
export interface InputProps extends InputHTMLAttributes<HTMLInputElement> {
  describedBy?: string
  id: string
}

define function Input(props: InputProps): ReactNode | null
```

### Props

The `InputProps` component accepts the following props:

| Name     | Default | Description                                                   |
| -------- | ------- | ------------------------------------------------------------- |
| describedBy     | ''   | The `id` attribute of the FieldMessage the Input is associated with. |
| id   | ''   |  An identifier that is shared with the Label `htmlFor` attribute.        |
