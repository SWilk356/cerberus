## As Prop

You can use the `as` prop to render the `Avatar` component as a different element.

<NoteAdmonition description="This is only recommended if you are replacing it with another image alternative (i.e., NextJS Image). If you want to make the avatar a link - simply wrap it in a link tag." />

<CodePreview preview={<NextAvatar />}>
```tsx title="next-avatar.tsx"
import { Avatar } from '@cerberus/react'
import Image from 'next/image'

function NextAvatar() {
  return (
    <Avatar
      as={
        <Image
          alt="Protector Cerberus"
          className={css({
            h: 'full',
            objectFit: 'cover',
            w: 'full',
          })}
          src="https://cerberus.digitalu.design/logo.svg"
          height={100}
          width={100}
        />
      }
      ariaLabel="With Next Image"
      size="lg"
    />
  )
}
```
</CodePreview>

## Customization

You can customize the avatar by utilizing the `cx` function.

<CodePreview preview={<CustomAvatar />}>
```tsx title="avatar.tsx"
import { Avatar } from '@cerberus/react'
import { css } from '@cerberus/styled-system/css'

function CustomAvatar() {
  return (
    <Avatar
      ariaLabel="Random image"
      className={css({
        bgColor: 'yellow.800',
        color: 'black',
        _hover: {
          bgColor: 'yellow.500',
        },
      })}
      src="https://source.unsplash.com/random/100x100"
    />
  )
}
```
</CodePreview>

## API

```ts showLineNumbers=false
export type AvatarImageProps = HtmlHTMLAttributes<HTMLImageElement> & {
  ariaLabel: string
  as?: never
  src: string
  width?: number
  height?: number
}
export type AvatarAsProps = {
  as: ReactNode
  ariaLabel?: never
  src?: never
  width?: never
  height?: never
}

export type AvatarProps = (HtmlHTMLAttributes<HTMLDivElement> &
  AvatarVariantProps) &
  (AvatarImageProps | AvatarAsProps)

define function Avatar(props: AvatarProps): ReactNode
````

### Props

The `Avatar` component accepts the following props:

| Name     | Default | Description                            |
| -------- | ------- | -------------------------------------- |
| ariaLabel     |    | The name of the avatar entity. |
| src     |    | The source of the image. |
| as    |    | A differnet element to render as the avatar. |
| height    |    | The height of the avatar. |
| width    |    | The width of the avatar. |
