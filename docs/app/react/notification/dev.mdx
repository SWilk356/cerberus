## Customization

For customizing the Notification, we recommend extending the `notification` slot recipe provided by the `@cerberus/panda-preset` package in your config.

## API

```ts showLineNumbers=false
export interface NotificationOptions {
  palette?: 'info' | 'success' | 'warning' | 'danger'
  heading: string
  description?: ReactNode
  onClose?: () => void
}

export interface NotificationProviderValue {
  notify: (options: NotificationOptions) => void
}

define function Notification(props: PropsWithChildren<{}>): ReactNode
````

### Show Method Options

The `notify` method accepts the following options:

| Name     | Default | Description                            |
| -------- | ------- | -------------------------------------- |
| palette | `info` | The palette variant used to theme the modal.           |
| heading | `''` | The heading of the modal. |
| description | `null` | The description of the modal. |
| onClose |  | An optional callback that runs when the user closes the notification. |
