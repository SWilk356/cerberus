 Spacing tokens can be used to determine the margin or padding of a element or group of elements. The spacing tokens are based on a 4px grid system.

 Tokens are available via the `spacing` object in the theme.

 <DripDivider />

 ## Values

| Name     | Value | Visual                                                    |
| -------- | ----- | --------------------------------------------------------- |
|  none    |   0   |  <SpacingVisual value="none" />             |
|  xs      |   0.25rem   |  <SpacingVisual value="xs" />                |
|  sm      |   0.5rem   |  <SpacingVisual value="sm" />                  |
|  md      |   1rem   |  <SpacingVisual value="md" />                |
|  lg      |   1.5rem   | <SpacingVisual value="lg" />                     |
|  xl      |   2rem   |  <SpacingVisual value="xl" />                     |
|  2xl     |   3rem   |  <SpacingVisual value="2xl" />                    |
|  3xl     |   4rem   |  <SpacingVisual value="3xl" />                   |
|  full    |   31.25rem | <SpacingVisual value="full" />           |

<DripDivider />

## Example Dev Usage

```jsx {4}
function Example() {
  return (
    <div className={css({
      px: 'spacing.xl',
    })}>
      <p>Some content</p>
    </div>
  )
}
```
