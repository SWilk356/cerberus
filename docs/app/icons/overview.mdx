# Overview

The Cerberus Icons library is a optional library to add visual elements to your React app.

## Installation

Install the Icons library of your choice. We recommend using the Carbon Icons library:

<BashTabs code="npm install @carbon/icons-react" />

## Usage

To use the Icons library, register it in your app by following the instructions in the [Customization section](../preset/customizing#option-2-cerberus-configuration).
